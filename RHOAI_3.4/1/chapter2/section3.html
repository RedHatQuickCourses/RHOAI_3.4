Automate Machine Learning Model Training with AutoML (Technology Preview)

What is it?

AutoML is an automated feature designed to streamline the end-to-end development of machine learning workflows for Tabular (classification and regression) and Time Series task types. It automatically handles feature engineering, hyperparameter optimization, and model selection by training several models and ranking them in a leaderboard.

Why does it matter?

Building scalable predictive intelligence is highly complex and typically requires specialized skills. AutoML matters because it:

  • Eliminates Manual Repetition: It handles the intricate steps of building a production-grade ML pipeline in one unified step.

  • Drives Precision: It evaluated models using advanced metrics such as confusion matrix, ROC curve, and F1 score to select the best possible solution for your use case.

  • Provides Deep Insight: Model Detail Views allow you to inspect optimized metrics specific to your task type, ensuring transparency in model selection.

  • Accelerates Productization: Reduces the time-to-market for predictive AI models by automating the transition from raw tabular data to trained artifacts.

How to use it?

  • Create Optimization Run: Upload your tabular or time-series data and configure your common and task-specific parameters.

  • Evaluate Results: Inspect the AutoML Evaluation Metrics and leaderboard to determine the best performing model.

  • Operationalize: Use the generated model artifacts and deployment notebook to run predictions or deploy the AutoML model for inference.
