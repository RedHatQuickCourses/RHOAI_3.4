= Adversarial Vulnerability Scanning with Garak Red Teaming (Technology Preview)

== What is it?

A tool for proactive adversarial vulnerability scanning included within the Llama Stack (LLS) distribution.

== Why does it matter?

It finds model jailbreaks before they reach production. It integrates with Chatterbox Labs to provide Model Risk Assessment data directly within the RHOAI Model Catalog.

== How to use it?

Utilize the Garak evaluation provider to generate security report artifacts and validate safety using Synthetic Data Generation (SDG).
